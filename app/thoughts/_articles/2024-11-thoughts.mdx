# 2024年11月碎碎记：在规则中突围，用AI重塑自我

> 永远热爱，永远不灭的心，哪怕一直一直都是不顺利的。

> 这个世界是属于勇敢者的。

十一月，是充满“杠杠”的一个月，关于消费与存款，关于他者之心，也关于做事的思路。条件在变，我们的应对策略也必须随之改变。

## ⛓️ 理解世界的“Lock-in”规则

我们很多人被牢牢地套牢在一个被系统性规定好的工作、生活、消费和财务系统以及价值观之中，无法摆脱。这就是所谓的 **"Lock-in"（锁死）** 概念，也是我们常说的“内卷化”的前提。

- **资本主义规则**: 由资产阶级制定，旨在保护其利益，将大多数人转化为满足这一目的的“社畜”。
- **消费主义与负债**: 通过广告等手段塑造价值观，导致人们不断提高消费并负债，从而必须努力工作以维持。
- **理财与退休系统**: 如社保、养老金等，这些系统假设个人无法管理投资，通过强制储蓄来控制个人的资金流动。

> 你是狼是羊？你在优秀，你还是羊；狼在弱，依然可以吃羊。弱肉强食，可怜无用。

要摆脱这种状态，**主动才有未来**，我们必须为自己负责，获取主动性，不走既定轨道。

## 🚀 个人成长：觉醒与突围之路

> 人什么时候真正开始成长？不是升职加薪，不是结婚繁殖，而是你开始觉醒、反思、质疑、审视……你所存在的环境以及自我。

- **搞钱才能通透**: 一个人只有赚到了钱，才会对这个世界的认知更加深刻，真正了解社会的运营机制。
- **别怕别人的嘴**: 害怕别人的议论，就什么事都别干，最终成为世俗需要你成为的样子。
- **信息差是捷径**: 善用信息差是最快的赚钱方式之一。

普通人想要创业成功，最大的问题有几点：
1.  **六边形战士**：一开始没钱、没资源，只能什么都自己干，从产品到流量，从私域到交付，每个环节都要亲自操盘。
2.  **持续找项目**：一直在找项目，而不是聚焦于一个项目深耕。高手赚钱是因为专注，而不是因为懂得多。
3.  **不懂得解放自己**：所有业务都自己做，无法标准化、流程化，无法把工作分给别人，导致公司无法做大。
4.  **不懂设置规则**：老板的规则是看项目、找人才、定团队、做激励、做分钱。你要把时间花在这。

## 🛠️ AI 工具与技术洞察

AI工具的价值在于它能成为我们的杠杆，放大我们的能力。

### AI Agent 与未来协作
- **AI Agent 的未来愿景**: 穆斯塔法提出，第一步是让协作助手能够“看见”用户眼中的一切，实现深度共情。这需要几年时间来实现稳定和精准的交互。
- **Google's Project IDX (Agent Mode)**: 被认为是一款能击败 Cursor, V0 & BOLT 的免费 AI 编辑器。
- **Devika**: 一个能理解高级人类指令，分解任务，并自主研究、编写代码以实现目标的AI软件工程师。
- **CopyCoder**: 一套利用 Next.js, shadcn UI, Cursor 和 AI 生成提示词的开发环境。

### AI 可视化与内容生成
- **核心 Idea**: 把信息变成信息图片/图表。AI 可以将文章内容转换成直观的图片，比如将复杂的技术架构描述直接生成为流程图。
- **相关工具**:
  - **Napkin.ai**: [napkin.ai](https://www.napkin.ai/)
  - **Infography**: [infography.in](https://infography.in/)

### 常用 AI 工具箱
- **信息获取与处理**:
  - **Claude**: 大模型王者，日常核心工具。
  - **Cursor**: 写代码王者。
  - **Gamma**: 制作 PPT 和文档的利器。
  - **v0**: 快速生成前端组件。
  - **Recraft**: 生成 UI 或线框图。
- **工作流自动化**:
  - **make.com**: 强大的无代码自动化平台。
  - **Coze / n8n / langflow**: 各有特色的工作流编排工具。
- **其他值得关注的项目**:
  - **lovable.dev**: 值得探索的新工具。
  - **MinerU**: PDF 提取工具。
  - **AIGC Github**: [GoogleCloudPlatform/generative-ai](https://github.com/GoogleCloudPlatform/generative-ai)
