# 2025年5月碎碎记：旷野、杠杆与AI

## 本月关键词：人生是旷野，不是轨道

本月反复思考的一个关键词是：**人生是旷野，不是轨道**。我们不应被预设的道路束缚，而应主动探索属于自己的可能性。最好的时机永远是现在，永远不要等。

主动去创造一个无网/无手机的环境，看看自己真正可以做什么。

## 关于心态与自我认知

### 核心认知

- **自发秩序**：无论什么样的秩序，都是自发秩序。不要推卸责任，所有的幸运，所有的苦难都是你的业报。
- **反脆弱**：在波动和不确定性中获益。
- **课题分离**：我不入你的局，你说什么都无所谓；不期待你的认可，那么你的评价和定义就不会影响我。我们各自做好自己的事，做到真正的“课题分离”。
- **接纳不完美**：用输出倒逼输入，接受不完美，持续迭代。过去的荣光不值一提。
- **恐惧的来源**：恐惧源于缺乏经验，而非无能。你害怕是因为你还没做过，而不是因为你做不到。缺乏经验才是需要解决的问题——而只有拥有行动的勇气才能解决它。

> 在这个世界上，你不用惧怕任何人。惧怕别人，本质上是因为你心存幻想。你怕老板，是因为你幻想老板能给你加薪。你怕有钱人，是因为你幻想有钱人能提携你一把。所以，你惧怕的不是对方，而是你对对方的幻想和妄念。禅宗说，无因，无事、无为。我无所依赖，自然无所畏惧。

> 大部分人在二三十岁上就死去了，因为过了这个年龄，他们只是自己的影子，此后的余生则是在模仿自己中度过，日复一日，更机械，更装腔作势地重复他们在有生之年的所作所为，所思所想，所爱所恨。
>
> ——罗曼·罗兰

### 情绪与欲望

- **情绪的层级**：底层发泄情绪，中层管理情绪，上层操控情绪。
- **情绪的价值**：情绪无用，专注下一个动作。失败不是终点，放弃才是。
- **欲望的挑战**：食欲很难克服，性欲极难克服，虚荣欲无法克服。
- **表达的本质**：人们最想表达的，永远不是内容本身，而是渴望被理解的心情。
- **获取的核心**：人们最想得到的，永远不是真正的价值，而是情感安慰。
- **说服的关键**：人们最容易被说服的，永远不是道理，而是利益。

## 关于商业、创业与搞钱

### 商业思维

- **问题即机会**：问题 = 机会，解决它就能赚钱。
- **市场历练**：你什么时候把自己丢到市场经济里，什么时候才算真正的成长。
- **核心能力**：开发和销售。
- **杠杆**：内容和代码是这个时代最重要的杠杆，大过资本和人力。
- **第一性原理**：
  1.  没有经济基础，谈不了“诗和远方”。
  2.  创业不是想做什么，而是市场需要什么。
  3.  多从客观角度分析问题。

> 成年人的世界只有一种硬通货：你手里能拿捏的资源，你能交换的筹码，你脚下踩实的土地。感情是虚的，利益是实的。你越早认清这一点，越早戒掉那些黏黏糊糊的情绪，越早学会用规则代替人情，用数字代替承诺，你就越能活得像个清醒的猎手，而不是猎物。

### 副业心态

1.  **目标导向**：直接以盈利为目标，不要把副业当兴趣爱好。
2.  **快速迭代**：快速行动，迭代优化，先让小规模盈利模式跑通。
3.  **行动为王**：行动大于思考，完美计划不如粗糙执行。不要担心外界评价，坚信自己的判断，专注把事情做出来。

### 搞钱逻辑

- **把自己当产品**：你得把自己当产品打磨。你了解客户需求多深？你能精准说出行业痛点吗？你有多少解决方案是“可落地”的？
- **短视频粉丝增长逻辑**：
  - `1k` 粉和 `1w` 粉的差距是**技术**，对短视频算法的理解。
  - `1w` 粉和 `10w` 的差距是**选题**，对流量网感的理解。
  - `10w` 粉和 `100w` 的差距是**工作量**，能不能发够一千条。
  - `100` 粉和 `1k` 粉的差距，是**懒**。

## 关于AI、技术与未来

### AI时代的机遇

- **一生一遇的机会**：AI 是我们这代人一生一遇级别的机会。
- **人机共生**：AI First，能用AI做的，就不用人做。搭建人机共生的共知平台。
- **未来的工作**：
    - 曾经的“简单任务”将不复存在，AI 会轻松取代。
    - 原来的“困难任务”会变成新的“简单任务”。
    - 曾经被认为“不可能的任务”将成为新的挑战。
- **AI的发展**：“AI 不可能恰好发展到让大家都很开心的那个点然后就停下来”。历史不会停下来，AI会一直发展，我们也应该一直发展自己，学会和现在以及未来的 AI 一起工作。

### 技术洞见

- **AI内容生产**：注意到 YouTube 上出现大量由 AI 生成的哲学、心理学频道，内容、声音、图片均为 AI 制作。这背后是掌握了工业化大规模快速生产高质量视频流程的制作者，商业模式是低成本获取流量赚取广告费。这会倒逼所有创作者提高产品质量，对消费者而言，就像从“家庭作坊”转向“工业化农场”，价格更便宜，质量有基本保证。
- **富文本编辑器**：对 Tiptap/ProseMirror 有深度实践经验。
- **关于 Agent 项目**，解决方案的三个方向：
    1. Prompt engineering + Workflow (如 ReAct)
    2. "协议 token" + 工具调用接口 + GRPO 微调
    3. RLVR + 系统级可靠性验证
    在一般情况下，难度、可靠性、潜在价值 `1 < 2 < 3`。

## 一些有用的工具与资源

- **SeleniumBase**: 一个全能型的浏览器自动化框架，核心是用 Python 包装了 Selenium 的 WebDriver API，功能强大，能写端到端测试、爬数据，甚至搞定 Cloudflare 防爬。
- **Morphik-Core**: 开源多模态 RAG 项目，专注于处理技术文档和多模态数据，构建 AI 应用。
- **Muscle Mem**: 开源的 AI Agent 缓存系统，通过存储和重用复杂行为，降低重复任务的计算成本。 ([Link](https://github.com/pig-dot-dev/muscle-mem/tree/main))
- **Awesome LangGraph**: 构建多角色、有状态应用的 LangGraph 资源宝库。 ([Link](https://github.com/von-development/awesome-LangGraph))
- **Basic Memory**: 通过与 AI 助手的对话构建持久语义图谱，将知识存储在本地 Markdown 文件中。 ([Link](https://github.com/basicmachines-co/basic-memory))
- **VidHub**: 家庭影院解决方案，类似 Infuse，支持多种云盘和服务器，开发者是国内的，更新很快。
- **Usertour**: 开源用户引导平台，轻松搞定应用内产品引导、检查清单和调查问卷。 ([Link](https://github.com/usertour/usertour))
- **BotBrowser**: 一款基于 Chromium 核心的隐身浏览器，能可靠绕过主流反爬虫检测。 ([Link](https://github.com/MiddleSchoolStudent/BotBrowser))
- **Anyquery**: 用 SQL 查询一切（GitHub、Notion 等40多个平台），还能让 LLMs 连接。 ([Link](https://github.com/julien040/anyquery))
- **Awesome-Agent-RL**: 解锁 Agent 强化学习潜力的资源库。 ([Link](https://github.com/0russwest0/Awesome-Agent-RL))
- **Wintoys**: 一款 Windows 优化软件。
