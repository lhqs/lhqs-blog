# 写作模板案例

Markdown模板， 提供高级的一些用法。

## 支持引用

> 人们总是低估 AI 发展遵循的“火车效应”：
> 一列远处开来的火车，你总觉得它很慢，直到它呼啸而过，人们才感受到速度，超过人类后，就再也看不见尾灯。


## 支持代码高亮

```jsx
// page.jsx
export default function Page () {
  return <h1>
    Hello, <User/>
  </h1>
}
```

## 支持tailwind语法
<div className="text-red-500">
通过“text-red-500”设置文本颜色为红色
</div>
