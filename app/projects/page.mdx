# 项目列表

如下项目是基于个人的需求进行开发。

## [网址导航](https://info.lhqs.ink/ '网址导航')

一个支持 [标签导航](https://info.lhqs.ink/) 与 [聚合搜索](https://info.lhqs.ink/search.html) 的个人站点. 现已收集三千多个精选网址. 主要解决 bookmark 管理的问题. 可以根据 tags 进行筛选，快速得到想要的网址.


## [四海创作系统](https://creator.lhqs.ink/ '四海创作系统')

四海创作系统 是一个内容创作者管理系统，提供用户认证、内容管理&分享、API集成等功能。该系统由现代化的前端和后端组成，适用于需要管理创作内容和API访问的应用场景。


## [极简博客](https://github.com/lhqs/lhqs-blog '极简博客')

一个极简、现代、内容驱动的个人博客系统，基于 Next.js 15、MDX、Shiki 和 Tailwind CSS 构建，支持高亮、数学公式、响应式设计和流畅的页面转场
